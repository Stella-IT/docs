## 준비물
* [Stella IT 계정](https://accounts.stella-it.com)
* [GitHub 계정](https://github.com)
* [Stella IT Code of Conduct 동의](https://opensource.stella-it.com/code_of_conduct)

## 기여 방법
1. Stella IT Docs 하단 GitHub 아이콘을 클릭하여 GitHub 저장소로 이동합니다.
2. 우측 상단의 Fork 버튼을 클릭하여 저장소를 포크합니다.
3. 포크된 저장소를 로컬 컴퓨터에 클론합니다.
4. 로컬에서 원하는 내용을 수정하거나 새로운 문서를 작성합니다.
5. 변경 사항을 커밋하고 푸시합니다.
6. 원본 저장소로 Pull Request를 생성합니다.
7. Stella IT 팀이 변경 사항을 검토하고 병합합니다.
